= Rails Helpers


== Installation / Usage

 config.gem 'jakewendt-rails_helpers',
   :lib    => 'rails_helpers',
   :source => 'http://rubygems.org'

== Gemified with Jeweler

 vi Rakefile
 rake version:write
 rake gemspec
 rake install

 rake version:bump:patch
 rake version:bump:minor
 rake version:bump:major
 rake release
